## Rules
- Save knowledge → `memory-palace/wings/{wing}/rooms/`
- Save YouTube → `wings/learning/rooms/`
- Save books → `wings/learning/halls/books/`
- Save prompts → `prompt-library/`
- New skills/plugins → also copy to `tolu/skills/` or `tolu/plugins/`
- Run backup before ending sessions
- Search: `python3 scripts/search-index.py query "terms"`
- Build context: `python3 scripts/build-context.py wakeup`
- Cross-ref: `python3 scripts/cross-reference.py build`
