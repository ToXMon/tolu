# Deep Research Synthesizer

## When to Use
Activate when asked to synthesize multiple sources, conduct deep research, identify patterns, or produce a comprehensive analysis from raw data.

## The Process

### Step 1: Gather Sources
- Collect all relevant documents, URLs, and data
- Assess source credibility (primary > secondary > tertiary)
- Note publication dates for recency

### Step 2: Extract and Filter
For each source:
- Extract key claims, findings, and data points
- Tag by theme/topic
- Filter out low-value or duplicative information
- Flag contradictions between sources

### Step 3: Identify Patterns
- Group findings by theme
- Find consensus points (agreed across sources)
- Identify disagreements or gaps
- Note emerging trends

### Step 4: Synthesize

```markdown
# Research Synthesis: [Topic]

## Executive Summary
[2-3 sentence overview of findings]

## Key Findings
### [Theme 1]
- [Finding with source citation]
- [Supporting evidence]

### [Theme 2]
- [Finding with source citation]

## Points of Consensus
- [What most sources agree on]

## Points of Debate
- [Where sources disagree and why]

## Gaps in Research
- [What's missing or under-explored]

## Implications
- [What this means for the reader]

## Sources
- [Numbered list of all sources]
```

## Constraints
- Always cite sources for every claim
- Distinguish between facts and interpretations
- Acknowledge uncertainty — don't overstate findings
- Minimum 3 sources for any synthesis
