# Social Content Strategy

## When to Use
Activate when asked to create ORIGINAL social content from scratch — X/Twitter threads, LinkedIn posts, video scripts, weekly calendars, or any platform-native content. This is the go-to skill for "write me an X thread about..." or "create a LinkedIn post on..."

## Philosophy
- **Test organically first**, boost winners ($10/day)
- **Find ONE format that works**, then scale it
- **Don't spread across platforms** until one works
- **Volume game**: Post daily, find what resonates
- **Consistency > perfection**: Ship daily, learn, double down

## Format Discovery
A "format" = repeatable content structure. Find one that works, then hammer it.

Known formats:
- "X things I learned from Y"
- Hot take → explanation → CTA
- Tutorial thread (1/N format)
- Behind-the-scenes of building
- Customer win stories
- Before/after transformations
- Meme templates for your niche
- Problem agitation → solution tease

## Content Pillars (Pick 3-4)
1. **Educational**: Teach your expertise
2. **Behind-the-Scenes**: Building in public
3. **Industry Commentary**: Hot takes, trends
4. **Entertainment**: Memes, humor, relatability
5. **Social Proof**: Customer wins, results

## Platform Specifications

### X/Twitter Thread
- Hook tweet: Pattern interrupt in line 1, stakes in line 2
- Body: One idea per tweet, numbered (1/N)
- Max 280 chars per tweet
- End with CTA tweet (follow, link, bookmark prompt)
- Use line breaks for readability
- Tone: Direct, punchy, opinionated

### X/Twitter Long-Form Article
- Title that doubles as a hook
- Opening paragraph: State the thesis boldly
- Subheadings every 3-5 paragraphs
- Mix bold claims with evidence
- End with a clear takeaway + CTA
- No character limit — use full depth

### LinkedIn Post
- Hook line 1: Stop the scroll (bold claim, question, or stat)
- Body: Short paragraphs (2-3 lines), use line breaks
- Professional but conversational tone
- Include 1-2 relevant hashtags (not 10+)
- End with a question or CTA to drive comments
- Avoid external links in post (hurts reach) — put in comments

### Video Script (Short-Form 30-60s)
- First 3 seconds: Hook (visual + verbal)
- One core message only
- Script structure: Hook → Problem → Solution → CTA
- Write for speaking, not reading (short sentences, natural pauses)
- Include visual cues [bracketed]
- End with clear next step

### Video Script (Long-Form YouTube)
- Hook in first 30 seconds (promise what they'll learn)
- Intro: Context + credibility
- Body: 3-5 sections with clear transitions
- Pattern: Make point → Show example → Give takeaway
- Outro: Summary + CTA (subscribe, link, next video)
- Include timestamps structure

## Complementary Skills
When creating content, consider layering in:
- **scqa-writing**: For structured argument flow (Situation → Complication → Question → Answer)
- **structured-copywriting**: For hooks, CTAs, and persuasion structure
- **tone-style-enforcer**: When output must match a specific brand voice
- **content-repurposing**: When asked to ALSO adapt the content to other formats

## Input Gathering
Before writing, collect (ask if not provided):
1. **Topic/angle**: What's this about?
2. **Audience**: Who's reading?
3. **Platform**: Where is this going?
4. **Goal**: Engagement, click-through, brand awareness, email signup?
5. **CTA**: What action should they take?
6. **Tone**: Casual, professional, provocative, educational?
7. **Content pillar**: Educational, BTS, commentary, entertainment, social proof?

## Output Format

### Single Post/Thread
```markdown
## [Platform] — [Content Type]
**Pillar**: [Educational/BTS/Commentary/Entertainment/Social Proof]
**Format**: [Format name]
**Goal**: [Engagement/CTR/awareness/signup]

---
[Full content ready to copy-paste]
---

**CTA**: [What action + where it leads]
**Suggested post time**: [Best time based on platform]
**Format notes**: [Why this format works for this content]
```

### Weekly Content Calendar
```markdown
# Content Calendar: Week of [Date]

## [Day] — [Platform]
**Type**: [Format]
**Pillar**: [Which pillar]
**Copy**:
[Full post]

---
[Repeat for each day]

## Format to Double Down On
[Based on engagement data, which format to scale]

## Metrics to Track
- Engagement rate per post
- Click-through to landing page
- Email signups from social
- Follower growth rate
```

## Constraints
- Every piece must have a clear CTA — no content without purpose
- Respect platform norms (character limits, tone, formatting)
- Content should feel native to the platform, not cross-posted
- If audience/topic not provided, ask before generating
- Flag which format type is being used so it can be tracked
- Suggest the best posting time when possible
