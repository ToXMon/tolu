# Structured Copywriting Skill

## When to Use
Activate when asked to write marketing copy, landing page content, sales emails, ad copy, or any conversion-focused text.

## The Copywriting Structure

### 1. Hook (The Opener)
- Start with a pattern interrupt — a bold claim, question, or surprising stat
- Must create an immediate "I need to read more" feeling
- Types: Pain hook, Curiosity hook, Story hook, Stat hook

### 2. Problem Agitation
- Name the reader's pain point clearly
- Show you understand the frustration
- Make the cost of inaction feel real

### 3. Solution Introduction
- Present the product/service/idea as the answer
- Keep it simple and specific
- Focus on benefits, not features

### 4. Proof & Credibility
- Social proof (testimonials, numbers, case studies)
- Authority signals
- "Don't just take my word for it" moments

### 5. Benefit Stacking
- List 3-5 clear benefits with micro-explanations
- Use "so you can..." language
- Prioritize by impact

### 6. Objection Handling
- Address the top 2-3 objections preemptively
- Use reframes, not arguments

### 7. CTA (Call to Action)
- Single, clear, specific action
- Create urgency without being slimy
- Make the next step obvious and easy

## Constraints
- Every sentence must earn its place — cut anything that doesn't persuade
- Write at a 6th-8th grade reading level
- Use short paragraphs (2-3 lines max)
- No jargon unless the audience is technical
