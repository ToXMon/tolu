# Content Repurposing Engine

## When to Use
Activate when asked to convert, repurpose, or adapt content from one format to multiple formats — e.g., turn a blog post into a thread, script, summary, or social posts.

## The Process

### Step 1: Analyze Source Content
- Identify the core message and key points
- Note the tone, audience, and intent
- Extract quotable lines, data points, and actionable insights

### Step 2: Select Target Formats
Choose from these output formats based on the request:
- **Twitter/X Thread** — Hook + numbered tweets + CTA
- **LinkedIn Post** — Professional tone, paragraph style
- **Newsletter Summary** — 3-5 key takeaways + link
- **Video Script** — Conversational tone with scene cues
- **Carousel Slides** — One idea per slide, visual-friendly text
- **Podcast Talking Points** — Bullet points with transition cues
- **Short-Form Video Script** — 30-60 sec, hook-first

### Step 3: Generate Each Format
For each selected format:
1. Apply format-specific structure (e.g., hook-body-CTA for threads)
2. Adapt tone and length to platform norms
3. Include relevant hashtags or CTAs
4. Maintain the core message across all versions

## Output Format
Present each format in a clearly labeled section:

```markdown
## [Format Name]
[Adapted content]

---

## [Format Name]
[Adapted content]
```

## Constraints
- Never lose the core message — every format must convey the same thesis
- Respect platform character limits
- Each format should feel native to its platform, not copy-pasted
