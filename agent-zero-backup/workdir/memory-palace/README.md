# 🏛️ Memory Palace — The Knowledge Vault

Organized knowledge repository covering books, YouTube content, references, and general knowledge.

## Structure

- **books/** — Book summaries, key references, and curated reading lists
- **youtube/** — Video transcripts, channel profiles, and playlist collections
- **references/** — Technical docs, academic papers, and tool documentation
- **knowledge/** — General knowledge, domain-specific expertise, and how-to guides

## Conventions

- Each file should be a self-contained markdown document
- Use descriptive file names: `topic-subtopic.md`
- Include source attribution and date where applicable
- Cross-reference related content with relative links
