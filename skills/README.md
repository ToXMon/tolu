# 🛠️ Custom Skills

Agent Zero custom skills following the SKILL.md standard.

## Adding a Skill

1. Create a directory: `skills/your-skill-name/`
2. Add `SKILL.md` with YAML frontmatter and instructions
3. Include any supporting scripts or templates

## Installed Skills

| Skill | Description |
|-------|-------------|
| [solidity](solidity/SKILL.md) | Production-grade Solidity standards — 32 best practices, Foundry workflows, testing patterns, CI pipeline (Cyfrin) |
| [battlechain](battlechain/SKILL.md) | BattleChain pre-mainnet L2 — deploy via battlechain-lib, Safe Harbor agreements, whitehat attacks, verification |
| [battlechain-tutorial](battlechain-tutorial/SKILL.md) | Interactive BattleChain deployment wizard — scans Foundry projects, 14 guided questions, generates all scripts |
| [karpathy-guidelines](karpathy-guidelines/SKILL.md) | Behavioral guidelines to reduce LLM coding mistakes |
| [here-now](here-now/SKILL.md) | Publish static sites instantly via here.now |
4. Also install to `/a0/usr/skills/` for active use
