# 📝 Prompt Library

A curated collection of prompts, templates, and techniques for LLM optimization.

## Structure

- **system-prompts/** — Full system prompt templates for agent behavior
- **task-prompts/** — Task-specific prompts (research, coding, analysis, etc.)
- **prompt-techniques/** — Techniques like chain-of-thought, few-shot, ReAct, etc.
- **templates/** — Reusable prompt templates with placeholders

## Usage

Each prompt file should include:
- Purpose and use case
- The prompt text (clearly marked)
- Expected input/output format
- Tips for customization
